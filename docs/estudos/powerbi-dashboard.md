# 📊 Power BI e Dashboards em Data Analytics

## 📖 O que é Power BI?

Power BI é uma ferramenta de Business Intelligence desenvolvida pela Microsoft, utilizada para:
- análise de dados
- criação de dashboards
- visualização de indicadores
- geração de relatórios interativos

A ferramenta é amplamente utilizada por empresas para apoiar tomadas de decisão orientadas por dados.

---

# 📈 O que são Dashboards?

Dashboards são painéis visuais que apresentam:
- métricas
- gráficos
- KPIs
- indicadores estratégicos

Seu objetivo é facilitar a interpretação de informações e o acompanhamento de resultados.

---

# 🧠 Importância da Visualização de Dados

A visualização de dados possui papel fundamental em Data Analytics porque permite:
- identificar padrões
- acompanhar desempenho
- facilitar análises
- comunicar informações com clareza

Uma boa visualização ajuda empresas a tomarem decisões mais rápidas e estratégicas.
