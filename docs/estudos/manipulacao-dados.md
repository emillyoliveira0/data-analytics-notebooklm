# 🧹 Manipulação e Limpeza de Dados

## 📖 Importância da Manipulação de Dados

A manipulação de dados é uma das etapas mais importantes em Data Analytics, pois os dados frequentemente apresentam:
- erros
- duplicidades
- valores ausentes
- inconsistências
- formatos diferentes

O tratamento adequado das informações contribui para análises mais confiáveis e estratégicas.

---

# 🔄 Processo de ETL

ETL significa:

- Extract (Extrair)
- Transform (Transformar)
- Load (Carregar)

Esse processo é utilizado para preparar dados antes das análises.

---

# 📥 Extract (Extração)

Consiste na obtenção dos dados a partir de diferentes fontes, como:
- bancos de dados
- planilhas
- APIs
- arquivos CSV
- sistemas empresariais

---

# 🔧 Transform (Transformação)

Etapa responsável por:
- limpar informações
- padronizar formatos
- remover inconsistências
- organizar os dados

Exemplos:
- remover duplicidades
- corrigir valores incorretos
- tratar dados nulos

---

# 📤 Load (Carregamento)

Consiste em enviar os dados tratados para:
- bancos de dados
- dashboards
- ferramentas analíticas
- sistemas de visualização

---

# 🧹 Limpeza de Dados

A limpeza de dados possui papel essencial para garantir qualidade nas análises.

---

# 📌 Principais Problemas Encontrados

## Dados Duplicados
Informações repetidas podem gerar análises incorretas.

---

## Valores Nulos
Campos vazios podem comprometer indicadores e relatórios.

---

## Dados Inconsistentes
Diferenças de formatos e padrões dificultam análises.

Exemplo:
- datas em formatos diferentes
- nomes escritos de formas variadas

---

# 🐍 Manipulação de Dados com Python

Bibliotecas como pandas são amplamente utilizadas para tratamento de dados.

### Exemplo de remoção de valores nulos

```python
dados.dropna()
```

---

### Exemplo de remoção de duplicidades

```python
dados.drop_duplicates()
```

# 📊 Impacto na Análise de Dados

- erros em dashboards
- indicadores incorretos
- análises inconsistentes
- decisões equivocadas

Por isso, a qualidade dos dados é considerada uma etapa fundamental em projetos analíticos.
