---

# 📊 Principais Bibliotecas

## pandas

Biblioteca utilizada para:
- manipulação de tabelas
- limpeza de dados
- análise de informações
- transformação de datasets

### Exemplo

```python
import pandas as pd

dados = pd.read_csv("vendas.csv")

print(dados.head())
```

---

## NumPy

Biblioteca utilizada para:
- operações matemáticas
- cálculos numéricos
- arrays multidimensionais

### Exemplo

```python
import numpy as np

valores = np.array([10, 20, 30])

print(valores.mean())
```

---

## Matplotlib

Biblioteca utilizada para criação de gráficos e visualizações.

### Exemplo

```python
import matplotlib.pyplot as plt

x = [1, 2, 3]
y = [10, 20, 30]

plt.plot(x, y)
plt.show()
```
