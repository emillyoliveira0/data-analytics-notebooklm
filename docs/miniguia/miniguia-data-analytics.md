# 📘 Mini Guia de Estudos — Data Analytics com NotebookLM

## 📖 Sobre o Projeto

Este mini guia foi desenvolvido como parte do desafio de projeto da DIO utilizando:
- NotebookLM
- engenharia de prompts
- curadoria de fontes
- documentação técnica no GitHub

O objetivo foi utilizar Inteligência Artificial como ferramenta de apoio aos estudos em Data Analytics, organizando conteúdos, resumos, prompts e insights relacionados à área de dados.

---

# 🎯 Objetivos de Estudo

Os principais objetivos deste projeto foram:

- compreender conceitos fundamentais de Data Analytics
- estudar KPIs e métricas
- explorar SQL para análise de dados
- entender dashboards e Power BI
- aprender conceitos de manipulação e limpeza de dados
- utilizar IA para organização e revisão de estudos
- aplicar engenharia de prompts no NotebookLM

---

# 📊 Principais Conceitos Estudados

## SQL para Analytics
SQL é utilizado para:
- consultas
- filtros
- agrupamentos
- análises de dados
- geração de relatórios

Principais comandos:
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- JOIN

---

## KPIs e Métricas
KPIs são indicadores utilizados para acompanhar desempenho e apoiar tomadas de decisão.

Exemplos:
- faturamento
- churn
- ROI
- ticket médio
- taxa de conversão

---

## Dashboards e Power BI
Dashboards permitem visualizar indicadores e métricas de forma estratégica.

O Power BI é amplamente utilizado para:
- visualização de dados
- relatórios
- Business Intelligence
- acompanhamento de KPIs

---

## Python para Data Analytics
Python pode ser utilizado para:
- manipulação de dados
- automação
- análise estatística
- visualização de informações

Principais bibliotecas:
- pandas
- NumPy
- Matplotlib

---

## Manipulação e Limpeza de Dados
A preparação de dados é uma etapa essencial em projetos analíticos.

Principais atividades:
- tratamento de valores nulos
- remoção de duplicidades
- padronização
- transformação de dados

---

# 📚 Glossário Resumido

| Conceito | Definição |
|---|---|
| KPI | Indicador-chave de desempenho |
| Dashboard | Painel visual de indicadores |
| ETL | Processo de extração, transformação e carregamento |
| SQL | Linguagem para consulta de dados |
| Data Analytics | Processo de análise de dados |
| Business Intelligence | Estratégia baseada em análise de dados |
