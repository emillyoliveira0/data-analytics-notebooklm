# 🧩 Prompts Iniciais — NotebookLM

## 📖 Objetivo

Este documento registra os primeiros testes de prompts realizados durante o desenvolvimento do projeto utilizando o NotebookLM como ferramenta de apoio aos estudos em Data Analytics.

O objetivo foi analisar:
- qualidade das respostas
- clareza das informações
- capacidade de síntese
- organização do conteúdo
- eficiência dos prompts

---

# 🤖 Prompt 1 — Introdução ao Data Analytics

## Prompt utilizado

```text
Explique o que é Data Analytics, seus principais objetivos e como ele é utilizado no mercado atualmente.
```

---

## Resultado Obtido

O NotebookLM apresentou:
- definição clara do conceito
- exemplos de aplicações práticas
- relação entre dados e tomada de decisão
- explicação introdutória acessível

---

## Pontos Positivos

- Linguagem objetiva
- Boa organização das informações
- Explicação fácil de compreender

---

## Pontos de Melhoria

A resposta poderia aprofundar:
- ferramentas utilizadas
- exemplos reais de mercado
- métricas e KPIs

---

# 🤖 Prompt 2 — KPIs e Indicadores

## Prompt utilizado

```text
Explique o que são KPIs em Data Analytics e apresente exemplos utilizados em empresas.
```

---

## Resultado Obtido

Foram apresentados indicadores relacionados a:
- vendas
- marketing
- desempenho
- conversão

---

## Aprendizados

Foi possível perceber que prompts mais específicos geram respostas:
- mais completas
- mais técnicas
- mais contextualizadas

---

# 🤖 Prompt 3 — Ferramentas de Data Analytics

## Prompt utilizado

```text
Liste as principais ferramentas utilizadas em Data Analytics e explique suas funções.
```

---

## Resultado Obtido

A IA apresentou ferramentas como:
- Python
- SQL
- Power BI
- Excel

Além disso, explicou:
- principais aplicações
- cenários de uso
- vantagens das ferramentas

---

# 📌 Observações Gerais

Durante os testes iniciais foi possível identificar que:
- prompts genéricos geram respostas superficiais
- prompts com contexto geram respostas mais completas
- a definição clara do objetivo melhora significativamente os resultados
