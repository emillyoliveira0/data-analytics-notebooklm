# ♻️ Prompts Reutilizáveis — Data Analytics

## 📖 Objetivo

Este documento reúne prompts reutilizáveis desenvolvidos durante os estudos de Data Analytics utilizando o NotebookLM.

O objetivo é facilitar:
- revisões futuras
- geração de resumos
- análise de indicadores
- organização de conteúdos
- extração de insights

Os prompts foram estruturados para apoiar estudos relacionados a:
- SQL
- KPIs
- dashboards
- Power BI
- Python
- análise de dados

# 💡 Principais Aprendizados

Durante os testes realizados foi possível perceber que:

# 📚 Conclusão

A criação de prompts reutilizáveis contribui para:
- organização dos estudos

Além disso, a engenharia de prompts se mostrou uma habilidade importante para potencializar o uso estratégico da Inteligência Artificial aplicada ao aprendizado.
