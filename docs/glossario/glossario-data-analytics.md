# 📘 Glossário de Data Analytics

## 📖 Objetivo

Este glossário reúne conceitos importantes relacionados à área de Data Analytics, visualização de dados, métricas, ferramentas analíticas e Inteligência Artificial aplicada aos estudos.

O objetivo é facilitar:
- revisões rápidas
- organização do conhecimento
- consulta de termos técnicos
- consolidação dos conceitos estudados

---

# 📊 Data Analytics

Processo de coleta, organização, tratamento e análise de dados com o objetivo de gerar insights e apoiar a tomada de decisão.

---

# 📈 KPI (Key Performance Indicator)

Indicador-chave utilizado para medir desempenho e acompanhar resultados estratégicos de um negócio ou processo.

Exemplos:
- faturamento
- churn
- ROI
- taxa de conversão

---

# 📉 Dashboard

Painel visual utilizado para apresentar métricas, gráficos e indicadores de forma organizada e interativa.

---

# 🧹 ETL (Extract, Transform, Load)

Processo utilizado para:
- extrair dados
- transformar informações
- carregar dados em outro sistema ou base

Muito utilizado em projetos de dados.

---

# 🗄️ SQL (Structured Query Language)

Linguagem utilizada para:
- consultar bancos de dados
- filtrar informações
- realizar análises
- manipular dados

---

# 🐍 Python

Linguagem de programação amplamente utilizada em:
- análise de dados
- automação
- machine learning
- visualização de dados

---

# 📊 Power BI

Ferramenta de Business Intelligence utilizada para:
- criação de dashboards
- visualização de dados
- análise de indicadores

---

# 📑 Dataset

Conjunto de dados utilizado para análises, estudos e construção de modelos analíticos.

---

# 📌 Churn

Indicador utilizado para medir perda ou cancelamento de clientes.

---

# 💰 ROI (Return on Investment)

Métrica utilizada para medir o retorno obtido sobre determinado investimento.

---

# 🛒 Ticket Médio

Indicador utilizado para medir o valor médio gasto por cliente em uma compra.

---

# 🔄 Data Cleaning

Processo de limpeza e tratamento de dados para remoção de:
- erros
- duplicidades
- inconsistências

---

# 📖 Storytelling com Dados

Prática de comunicar informações utilizando:
- contexto
- visualizações
- métricas
- narrativa estratégica

---

# 🤖 Engenharia de Prompt

Processo de criação e refinamento de instruções utilizadas para interação com modelos de Inteligência Artificial.

---

# 🧠 Inteligência Artificial (IA)

Tecnologia capaz de realizar tarefas que normalmente exigiriam inteligência humana, como:
- análise
- interpretação
- geração de conteúdo
- identificação de padrões

---

# 📌 Observações

O glossário será atualizado continuamente conforme novos conceitos e ferramentas forem estudados ao longo do projeto.
