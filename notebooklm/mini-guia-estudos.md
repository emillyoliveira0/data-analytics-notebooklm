# 📊 Mini Guia de Estudos — Data Analytics

## 📖 O que é Data Analytics?

Data Analytics é o processo de coleta, organização, tratamento e análise de dados com o objetivo de gerar insights e apoiar a tomada de decisão.

A análise de dados é utilizada por empresas para:
- identificar padrões
- acompanhar métricas
- melhorar processos
- reduzir custos
- aumentar resultados

---

# 🧠 Principais Tipos de Análise de Dados

## Análise Descritiva
Foca em entender o que aconteceu por meio dos dados.

Exemplo:
- relatórios
- dashboards
- métricas de vendas

---

## Análise Diagnóstica
Busca entender o motivo de determinado resultado.

Exemplo:
- identificar causas de queda nas vendas
- análise de desempenho

---

## Análise Preditiva
Utiliza dados históricos para prever tendências futuras.

Exemplo:
- previsão de demanda
- previsão de churn

---

## Análise Prescritiva
Sugere ações com base nos dados analisados.

Exemplo:
- recomendações automáticas
- otimização de processos

---

# 🛠️ Ferramentas Utilizadas em Data Analytics

## Python
Muito utilizado para:
- manipulação de dados
- automação
- análise estatística
- visualização de dados

Bibliotecas populares:
- pandas
- numpy
- matplotlib

---

## SQL
Linguagem utilizada para:
- consultar dados
- filtrar informações
- realizar análises em bancos de dados

Principais comandos:
- SELECT
- WHERE
- GROUP BY
- JOIN

---

## Power BI
Ferramenta utilizada para:
- criação de dashboards
- visualização de indicadores
- análise interativa de dados

---

## Excel
Muito utilizado para:
- análises rápidas
- organização de dados
- tabelas dinâmicas
- gráficos

---

# 📈 KPIs e Indicadores

KPIs (Key Performance Indicators) são métricas utilizadas para medir desempenho.

Exemplos:
- faturamento
- ticket médio
- taxa de conversão
- churn
- ROI
- CAC

---

# 🔄 Fluxo Básico da Análise de Dados

## 1. Coleta de Dados
Obtenção das informações.

## 2. Limpeza de Dados
Tratamento de erros, duplicidades e inconsistências.

## 3. Transformação
Organização e preparação dos dados.

## 4. Análise
Extração de insights.

## 5. Visualização
Criação de gráficos e dashboards.

## 6. Tomada de Decisão
Uso estratégico das informações analisadas.

---

# 📊 Storytelling com Dados

Storytelling com dados é a prática de comunicar informações de forma clara e estratégica utilizando:
- gráficos
- dashboards
- indicadores
- contexto de negócio

O objetivo é facilitar a interpretação dos dados e apoiar decisões.
